Binds a PHP variable to a corresponding named or question mark placeholder in the SQL statement that was use to prepare the statement. Unlike PDOStatement::bindValue(), the variable is bound as a reference and will only be evaluated at the time that PDOStatement::execute() is called. Returns a boolean value indicating success.

Parameters:
int$posParameter identifier (for determining what to replace in the query).
mixed$valueThe value to bind to the parameter.
int$typeExplicit data type for the parameter using the PDO::PARAM_* constants. Defaults to PDO::PARAM_STR.
int$lengthLength of the data type. To indicate that a parameter is an OUT parameter from a stored procedure, you must explicitly set the length.
