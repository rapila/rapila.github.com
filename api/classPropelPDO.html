PropelPDO::prepare ( $  sql,
$  driver_options = array() 
)

Prepares a statement for execution and returns a statement object.

Overrides PDO::prepare() in order to:

  • Add logging and query counting if logging is true.
  • Add query caching support if the PropelPDO::PROPEL_ATTR_CACHE_PREPARES was set to true.
Parameters:
string$sqlThis must be a valid SQL statement for the target database server.
arrayOne or more key => value pairs to set attribute values for the PDOStatement object that this method returns.
Returns:
PDOStatement
