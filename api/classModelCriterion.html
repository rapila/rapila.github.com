Figure out which MocelCriterion method to use to build the prepared statement and parameters using to the Criterion comparison and call it to append the prepared statement and the parameters of the current clause. For performance reasons, this method tests the cases of parent::dispatchPsHandling() first, and that is not possible through inheritance ; that's why the parent code is duplicated here.
