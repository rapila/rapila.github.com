static StringPeer::getStaticStrings (   $sLanguageId)
static

Loads all the static strings from either the cache or the ini files. Note that ini files in modules are not verified for outdatedness, so if they were updated, just clear all the caches or hard reload a page This method should not be called directly from outsite StringPeer except for testing and debugging purposes
