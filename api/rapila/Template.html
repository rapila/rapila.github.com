Class Template

Description
Description | Vars (details) | Methods (details) Constants (details)

class Template is used to manage building a tree with static template texts and dynamic identifiers that have the form of

  1. {{identifier=value;param=value}}
. Those can have special meaning (SpecialTemplateIdentifierActions) and be replaced by the Template class or can be replaced by the user of the template using replaceIdentifier() or replaceIdentifierMultiple().

All replaceIdentifier… methods take a flag parameter. The possible flags can be bitwise ORed together. The flags are described in the constants section. You can also provide a new template with some default flags. All Templates whose file name start with “e_mail_” will automatically get the NO_HTML_ESCAPE flag while those ending in .css.tmpl or .js.tmpl will automatically get NO_HTML_ESCAPE|ESCAPE.
