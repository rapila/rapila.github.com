		Constructor __construct (line 18)
	
 
	
	

Used to create FileResource instances. ResourceFinder will call this with given full path, instance prefix, relative path;  other uses will likely let the function figure out instance prefix and relative path.


Setting relative path to non-null and instance prefix null is not allowed.  Setting instance prefix but not relative path is redundant.
