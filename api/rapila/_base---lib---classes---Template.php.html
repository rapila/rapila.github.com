Class Description
Template class Template is used to manage building a tree with static template texts and dynamic identifiers that have the form of
  1. {{identifier=value;param=value}}
. Those can have special meaning (SpecialTemplateIdentifierActions) and be replaced by the Template class or can be replaced by the user of the template using replaceIdentifier() or replaceIdentifierMultiple().
