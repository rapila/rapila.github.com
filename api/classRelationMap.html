RelationMap::getColumnMappings ( $  direction = RelationMap::LOCAL_TO_FOREIGN)

Get an associative array mapping local column names to foreign column names The arrangement of the returned array depends on the $direction parameter:

  • If the value is RelationMap::LOCAL_TO_FOREIGN, then the returned array is local => foreign
  • If the value is RelationMap::LEFT_TO_RIGHT, then the returned array is left => right
Parameters:
int$directionHow the associative array must return columns
Returns:
Array Associative array (local => foreign) of fully qualified column names
